# 📞 Call Centre Analytics Dashboard – Excel

An interactive, data-driven Call Centre performance dashboard built entirely in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers.

## 📊 Overview
This dashboard tracks key call centre KPIs across three regions — **Cincinnati, Cleveland, and Columbus** — helping monitor agent performance, customer satisfaction, and call trends at a glance.

## 🔑 Key Features
- **KPI Cards**: Total Calls (1,000), Total Amount (₹96,623), Duration (89,850 mins), Avg. Rating (3.9), Happy Callers (307)
- **Call Trend Chart**: Month-wise call volume trend (Jan–Dec)
- **Day-wise Call Distribution**: Bar chart showing calls by weekday (Mon–Sun)
- **Female vs Male Analysis**: Stacked bar chart comparing gender-wise call distribution by region
- **Rating Distribution**: Histogram of customer ratings (1–5 scale)
- **Representative Performance**: Slicer-controlled bar charts for Calls, Duration, and Amount per representative (R01–R05)
- **Detailed Pivot Table**: Region and customer-wise amount breakdown across representatives with Grand Total

## 🛠️ Tools & Techniques Used
- Pivot Tables & Pivot Charts
- Slicers for interactive filtering
- Conditional formatting (data bars)
- Dashboard layout design with KPI cards
- Excel formulas (SUM, AVERAGE, COUNTIF etc.)

## 🎯 Purpose
Built as a hands-on data analytics project to practice Excel dashboarding skills — data visualization, KPI tracking, and interactive reporting — as part of my AI/Data journey.
